Sonoma Food Inspection Analytics

  1. Home
  2. Project Work

Project information

  • Category: Data Analytics/Science
  • Project date: Sept 2022 - Dec 2022
  • GitHub Code Power BI Dashboard

Data Analysis & ETL Pipeline Implementation:

  • Analyzed food inspection data from multiple regions, including Sonoma County, Austin, New York, and Chicago, focusing on the layout, fields, and relationships between data elements to understand the organization of the datasets.
  • Modeled the dataset into a Datamart and managed it via Azure Data Studio, performing ELT operations using Talend ETL to efficiently process and transform the data for analysis.

Execution Highlights:

  • Created dashboards in Tableau and PowerBI to visualize trends in food inspections, focusing on time-based analysis (year, quarter, month, and day). along with exploring various inspection criteria, such as pass vs fail rates, violation categories (major/minor/other), and restaurant inspection frequency.
  • Mapped geographical inspection data to highlight areas with the highest number of inspections and the most violations, providing actionable insights for health authorities and businesses.
